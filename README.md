# 扫雷

一个基于原生 HTML、CSS、JavaScript 的扫雷小游戏原型。

## 当前版本

- 默认难度为 9 x 9，10 个地雷
- 左键翻开格子
- 右键插旗或取消插旗
- 首次点击保证安全
- 自动展开周围空白区域
- 踩到地雷判负，找出全部非地雷格子判胜

## 本地运行

直接用浏览器打开 `index.html` 即可开始游戏。

如果你想用本地服务运行，也可以执行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`
